## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ cp .env.example .env
$ yarn docker:up
$ yarn seed
$ yarn start:dev

Swagger docs located at: localhost:8000/api

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

```
